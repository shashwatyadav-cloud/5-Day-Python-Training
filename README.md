# 🐍 Day 1 – Python Training (5-Day Program)

Welcome to **Day 1** of the 5-day Python training program. This repository contains beginner-friendly Python exercises focused on fundamental concepts like variables, input/output, conditionals, and string manipulation.

---

## 📂 Repository Contents

### ✅ Practice Questions

**Q1.** Write a program to swap two variables without using a third variable.

**Q2.** Create a program that takes user input for name and age, then prints a formatted message using f-strings.

**Q3.** Write a program to check if a number is even or odd using the modulo operator.

**Q4.** Convert temperature from Celsius to Fahrenheit and vice versa.

**Q5.** Check if a given string is a palindrome (case-insensitive).

**Q6.** Count the frequency of each character in a string using a dictionary.

**Q7.** Write a program to format a string using all three formatting methods:

* `%` formatting
* `.format()` method
* f-strings

**Q8.** Write a program to check if a year is a leap year.

**Q9.** Check if two strings are anagrams of each other.

**Q10.** Create a CLI calculator that performs basic arithmetic operations based on user input with proper input validation.

## 🚀 Getting Started

1. Install Python (version 3.x recommended)
2. Clone the repository:

   ```bash
   git clone <your-repo-link>
   ```
3. Navigate to the project folder:

   ```bash
   cd <repo-name>
   ```
4. Run any Python file:

   ```bash
   python filename.py
   ```
